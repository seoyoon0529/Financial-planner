# 재무 계획 웹앱 PRD (Cursor 참고문서용)

## 1. 개요
HTML, CSS, JavaScript만 사용하고 **백엔드 없이 LocalStorage**로 데이터가 저장되는 경량 재무 관리 웹앱.  
수입/지출 기록, 카테고리 관리, 지출 패턴 분석, 필터·정렬, 알림 기능을 제공한다.

## 2. 목표
- 설치나 로그인 없이 재무 기록·분석 기능 제공  
- LocalStorage 기반의 데이터 영속성 확보  
- 간단하고 빠른 UI로 사용자 경험 개선  
- 사용자의 소비 패턴을 분석해 재무 계획 수립 지원  

## 3. 주요 사용자
- 대학생 및 사회초년생 등 간단한 가계부가 필요한 사용자  
- 회원가입 없이 빠르게 사용하고 싶은 사용자  
- 지출 패턴을 기본 수준에서 분석하고 싶은 사용자  

## 4. 기능 요구사항

### 4.1 수입/지출 기록 관리
- CRUD 기능 제공  
- 입력 항목: 날짜, 구분, 카테고리, 금액, 메모  
- 저장 키: `transactions`

### 데이터 예시
```json
{
  "id": "uuid",
  "type": "expense",
  "category": "식비",
  "amount": 8000,
  "memo": "점심",
  "date": "2025-11-28"
}
```

---

### 4.2 카테고리 관리
- 기본 카테고리 제공  
- 사용자 정의 추가/수정/삭제  
- 사용 중인 카테고리 삭제 처리 규칙 필요  
- 저장 키: `categories`

---

### 4.3 지출 패턴 분석
- 월별 합계, 카테고리 비율, 최근 7일 추세  
- Chart.js 또는 Canvas 사용  
- 월 필터, 카테고리 그룹화, 날짜 기반 분석 로직 포함

---

### 4.4 필터 및 정렬
- 필터: 날짜, 구분, 카테고리, 금액  
- 정렬: 날짜순, 금액순, 카테고리 순  

---

### 4.5 알림 기능
- 월 지출 한도 초과  
- 카테고리 한도 초과  
- 고정 지출일 알림  
- 저장 키: `settings`

---

## 5. 비기능 요구사항
- LocalStorage 기반의 빠른 응답  
- PC/모바일 반응형 구성  
- 데이터는 브라우저 내에서만 유지  

---
